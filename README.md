# a demo to identify resistors on PCB(not finished yet)

更新：
- 程序能够完成的功能：在PCB上定位疑似色环电阻的物体，并通过SVM分类得到真的电阻，
通过图像处理提取色环

- 待完成：通过分析提取到的色环图像得到电阻阻值


运行环境：
- JDK8
- IntelliJ IDEA or Eclipse
- OpenCV 3.2
